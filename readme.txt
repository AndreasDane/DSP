Neural Network Football Ranking System

Edited: 22 April 2020
Author: Andreas Dane 

Features:

* Ranks EURO 2016 teams using a neural network. 
* Match prediction based on FIFA and neural network rankings. 

Modules: 

* Python 3.6.8
* Keras 2.3.1
* TensorFlow 2.1.0

Instruction: 

* Run main.py file to initiate system.
